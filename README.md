# Warriors – Into the Wild

Browser game based on the novel series by Erin Hunter.

**Play:** https://karolkarolasia.github.io/cursor/

## Controls

| Action | Keys |
|---|---|
| Move left | ← / A / J |
| Move right | → / D / L |
| Jump | ↑ / W / I / Space |
| Sneak | ↓ / S / K |
| Interact / talk | E / Enter |

## Roadmap

- **Chapter I** *(current)* — engine prototype: Rusty at Twoleg Place, forest edge, Graypaw encounter, Bluestar
- **Chapter II** — ThunderClan camp interior, training with Tigerclaw
- **Chapter III** — first hunting assessment, ShadowClan border
- ...
